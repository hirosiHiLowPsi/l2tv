L2TV 使い方説明書 / 安全性について
====================================

重要なお知らせ: アンインストールと7z版の使い方
----------------------------------------------

- 7z版を解凍すると、構成データをまとめた「L2TV」フォルダが自動で作成されます。事前に空のフォルダを作る必要はありません。
- 解凍後に作成された「L2TV」フォルダ内の L2TV.exe を起動してください。
- 7z版はインストール不要です。削除するときは、Windowsのアンインストーラーを使わず、解凍したL2TV専用フォルダをゴミ箱へ移動してください。
- 削除してよいのは、解凍して作ったL2TV専用フォルダだけです。デスクトップやダウンロードフォルダなど、親フォルダそのものを削除しないでください。
- 1.0.1以前のインストーラー版で、デスクトップなどの個人フォルダをインストール先に選んだ場合は、アンインストーラーを実行しないでください。

L2TV は「LR2 Table Viewer」の略です。
Lunatic Rave 2 のローカルDBとBMS難易度表を読み込み、
登録譜面のクリアランプ、スコア、BP、ライバル比較などを
見やすく確認するためのツールです。

このツールは LR2 のデータを見るためのアプリです。
LR2 の score.db / song.db / Rival DB を書き換えたり削除したりしません。


1. このツールでできること
------------------------

基本機能:

- LR2 の score.db を読み込めます。
- LR2 の song.db を読み込めます。
- LR2 の Rival フォルダを読み込めます。
- score.db仕様を 自動判別 / 従来LR2IR互換 / StellaverseIR / BMS-IR から選べます。
- プレイヤー名を表示できます。
- SP段位を表示できます。
- st段位 / sl段位を表示できます。
- 難易度表を読み込んで、表に登録されている譜面だけを集計できます。
- 難易度表を読み込まなくても、Player Data だけ表示できます。

難易度表:

- Stella
- Satellite
- 発狂BMS難易度表
- Overjoy
- Dystopia
- 難易度表一覧から選択した表
- 読み込んだ難易度表の表示 / 非表示切り替え
- 読み込み済みの難易度表は、URLではなく表タイトルで表示

Player Data:

- プレイヤー名
- ID
- FORCE RATE と称号バッジ
- FORCE RATE は発狂BMS難易度表、初代Overjoy、第二期Overjoyの譜面定数をMD5で照合して算出
- 譜面定数はLR2IR Archive上の「クリア人数に対するAAA到達人数の割合」を主軸にしたスコア力重視の定数
- 単曲FORCE = 譜面定数 × スコア係数
- クリアランプ係数とBEST20補正は使用せず、単曲FORCE上位50譜面と最高合格GENOSIDE2018 SP段位1件を合わせた最大51対象の単純平均で算出
- 発狂段位定数はLR2IR Archive上の「プレイ人数に対する合格+AAA達成人数の割合」を基準に算出
- 発狂段位コースはEXスコアが取得できる場合、段位定数にもスコア係数を反映。GENOSIDE2018 Overjoyはスコア係数を掛けず、合格時点で段位定数満額
- SP段位
- SP段位の正式名称
- st段位 / sl段位
- st/sl段位の正式名称
- Overjoy などの特殊段位表示
- st12 / Overjoy などの虹色アニメーション表示
- 今回の打鍵回数表示
- score.db にプレイ時間項目がある場合、プレイ時間表示

ランプ集計:

- CLEAR LAMP グラフ
- SCORE LAMP グラフ
- レベル別の横棒グラフ
- グラフ内のパーセンテージ表示
- ランプ内訳
- スコア内訳
- レベルフォルダごとのランプ / スコア内訳表示
- レベルフォルダの達成状況に応じた色付け

譜面一覧:

- レベル別に譜面一覧を表示
- Lv / Title / Artist / Lamp / Rank / EX/Rate / BP / Play Count / Rival を表示
- Rank は AAA / AA / A などを色付きバッジで表示
- Lv は見出しのみ表示
- Title / Artist などで並べ替え
- 曲名とアーティスト以外を中央揃えで表示
- 表のシンボル付きレベル表示
- クリアランプに応じた行背景色
- ランプ表記を短縮表示
- Rival比較表示
- BP表示
- NO PLAY の譜面は BP を NO Data 表示
- song.db に存在しない譜面は NS 表示
- レベルフォルダを開いている間は Folder Close ボタンで下部から閉じられます。
- レベルフォルダを開いている間は Folder Option ボタンから小型の並べ替えヘッダーを表示できます。

本日更新:

- 前回読み込みからクリアランプが更新された譜面を表示
- オプションで、クリアランプに関係なくBPが減った譜面も表示
- スコアが更新された譜面を「スコア更新」欄に表示
- ランプ更新やBP更新と同時にスコアが伸びた譜面も、スコア更新欄に表示
- 今日の更新を画像出力
- 今回の打鍵数を表示
- score.db にプレイ時間項目がある場合、プレイ時間を表示
- 更新譜面数を表示

RIVAL:

- LR2 の Rival フォルダを読み込み
- ライバルごとのクリアランプを比較
- ライバルごとのスコア勝敗を比較
- 譜面一覧の Rival 列に相手スコア / レート / ランプを表示
- ライバルごとの WIN / LOSE 数を表示
- WIN / LOSE の割合を黄色 / 青のグラフで表示
- NO PLAY は負け数に含めません
- 比較対象のライバルをチェックボックスで選択
- 全選択 / 全解除
- 勝敗表示を「全譜面」または各難易度表ごとに切り替え
- WIN率順 / LOSE率順 / 名前順でライバル一覧を並べ替え

画像出力:

- 本日更新の画像出力
- 難易度表サマリーの画像出力
- CLEAR LAMP / SCORE LAMP の画像出力
- スクショ保存先を自由に指定可能
- 保存完了時に「スクショ保存完了！」と表示

表示設定:

- 日本語 / English の表示言語切り替え
- Light Aqua テーマ
- Geek Dark テーマ
- CLEAR LAMP / SCORE LAMP のワンタッチ切り替え


2. 必要なもの
-------------

- Windows PC
- Lunatic Rave 2 本体
- LR2 の score.db
- LR2 の song.db

あると便利なもの:

- LR2 の Rival フォルダ
- インターネット接続

一般的な配置例:

score.db:
  LR2files\Database\Score\プレイヤー名.db

song.db:
  LR2files\Database\song.db

Rival フォルダ:
  LR2files\Rival

環境によって保存場所は異なります。
わからない場合は LR2 フォルダ内の LR2files を探してください。


3. 起動方法
-----------

7z 版の場合:

1. L2TV-2.0.1-win-x64.7z を任意の場所に解凍します。
2. 自動で作成された「L2TV」フォルダ内の L2TV.exe を実行します。
3. インストールは不要です。

4. 初回の使い方
---------------

1. L2TV を起動します。
2. 画面右上の「メニュー」を開きます。
3. 「LR2 score.db パス」の参照ボタンから score.db を選びます。
4. 「song.db パス」の参照ボタンから song.db を選びます。
5. score.db仕様を選びます。迷う場合は「自動判別」のままで大丈夫です。
6. 必要なら「Rival フォルダ」の参照ボタンから Rival フォルダを選びます。
7. 「一覧を開く」から難易度表一覧を開き、読み込みたい表にチェックを入れます。
8. 「表とランプを読み込む」を押します。

難易度表を選ばずに読み込んだ場合でも、Player Data は表示できます。

score.db仕様:

- 自動判別: 190114.db のような数字名のDBは StellaverseIR、114514boy.db のような名前入りDBは従来LR2IR互換として扱います。
- 従来LR2IR互換: 既存のLR2IR互換DBや、114514boy.db 形式を使うIR向けです。
- StellaverseIR: score.db 内のIDを使ってStellaverseIRのプレイヤー名取得を試みます。取得できない場合はDB内の名前を使います。
- BMS-IR: BMS-IR形式のDBとして読み込みます。段位情報はDBとsong.dbからオフラインで推定します。


5. メニューで設定できること
--------------------------

読み込み設定:

- score.db を指定
- song.db を指定
- score.db仕様を選択
- スクショ保存先を指定
- Rival フォルダを指定
- 難易度表一覧を別ウィンドウで開く
- 難易度表一覧から読み込む表を選択
- General / Personal / Self-made Chart Only で難易度表一覧を絞り込み

表示設定:

- 表示言語を 日本語 / English から選択
- 初回起動時は表示言語を選択するメッセージが表示されます。
- テーマを Light Aqua / Geek Dark から選択
- 本日更新に BP更新も含めるか選択

保存:

- 入力したパスや設定は保存され、次回起動時に復元されます。
- score.db / song.db / Rival フォルダのパスも次回に引き継がれます。


6. 難易度表について
------------------

「一覧を開く」から難易度表一覧を開き、チェックした表を読み込みます。
一覧は「一覧を更新」ボタンで再取得できます。
表名検索と General / Personal / Self-made Chart Only のタグフィルターで、必要な表だけを探せます。
読み込み後は、URLではなく難易度表タイトルで表示されます。


7. 表示の見方
-------------

Player Data:
  プレイヤー名、ID、SP段位、st/sl段位などが表示されます。
  GENOSIDE2018段位コースの合格判定は、EASY CLEAR を含めず NORMAL CLEAR 以上を合格として扱います。
  段位合格は score.db に保存された段位認定コース自体のクリア記録だけで判定し、構成する単曲のクリアランプは参照しません。
  段位情報はDBとsong.dbからオフラインで推定します。IRサイトから段位を補完取得しません。

ランプ内訳:
  難易度表全体のクリアランプ数を表示します。

スコア内訳:
  AAA / AA / A / B / C / D / E / F / NP / NS の数を表示します。
  NO PLAY の譜面は F として数えず、NPとして扱います。

CLEAR LAMP:
  FC / HC / NC / EC / FL / NP / NS の割合をレベル別に表示します。

SCORE LAMP:
  AAA / AA / A / B / C / D / E / F / NP / NS の割合をレベル別に表示します。

譜面一覧:
  レベルを開くと、譜面ごとの詳細を確認できます。

Rank:
  譜面一覧で AAA / AA / A などのスコアランクを表示します。

BP:
  BAD + POOR の合計です。NO PLAY の譜面は NO Data と表示します。

NS:
  NO SONG の略です。song.db に曲データが見つからない譜面です。

Folder Option:
  レベルフォルダを開いている間だけ表示されます。
  押すと小型の並べ替えヘッダーを表示します。
  表示中はボタン名が Option Close になります。

Folder Close:
  レベルフォルダを開いている間だけ表示されます。
  押すと開いているレベルフォルダを閉じます。

Lamp Updates:
  前回読み込みと比べて更新された譜面を確認できます。
  メニューの「表外譜面も更新楽曲として表示する」をONにすると、読み込んでいる難易度表に含まれない譜面も比較対象になります。
  表外譜面は、前回読み込み時に保存したローカルスコア状態と現在の score.db を比較して表示します。

RIVAL:
  読み込んだライバルとのスコア勝敗を確認できます。


8. RIVAL機能について
--------------------

RIVALボタンからライバルメニューを開けます。

できること:

- 読み込んだライバル一覧を表示
- ライバルごとのスコア数を表示
- チェックボックスで比較対象をON/OFF
- 全選択
- 全解除
- WIN / LOSE 数を表示
- WIN / LOSE 率をグラフ表示
- 勝敗集計の対象を「全譜面」または各難易度表ごとに切り替え
- WIN率順 / LOSE率順 / 名前順で並べ替え

勝敗判定:

- 自分のEXスコアが高い場合は WIN
- 自分のEXスコアが低い場合は LOSE
- 同点は DRAW と表示されます
- 自分または相手が NO PLAY / NO SONG の場合は勝敗に含めません

譜面一覧の Rival 列:

- 相手ランプに応じた背景色
- WIN / LOSE / DRAW
- ライバル名
- 相手EXスコア
- 相手スコアレート
- 相手ランプ


9. 本日更新について
-------------------

本日更新では、前回読み込み時の状態と今回読み込み時の状態を比較します。

表示されるもの:

- クリアランプが更新された譜面
- BP が減った譜面
- EXスコアが伸びた譜面

BP更新:
  メニューの「本日更新にBPが減った譜面も表示する」をONにすると表示されます。

スコア更新:
  クリアランプやBP更新とは別に、EXスコアが伸びた譜面を表示します。
  ランプ更新と同時にスコアが伸びた譜面もスコア更新欄に表示されます。

画像出力:
  Lamp Updates にある「今日の更新を画像出力」から出力できます。


10. 画像出力について
-------------------

本日更新の画像出力:

- Lamp Updates にある「今日の更新を画像出力」から出力できます。
- ランプ更新、BP更新、スコア更新を画像にできます。
- 今回の打鍵数、プレイ時間、更新譜面数も表示されます。
- 更新譜面数が非常に多い場合は、250件ごとに複数PNGへ自動分割されます。

難易度表サマリーの画像出力:

- 各難易度表のサマリー画面から出力できます。
- ランプ内訳、スコア内訳、CLEAR LAMP / SCORE LAMP グラフを出力できます。

保存先:

- メニューの「スクショ保存先」で自由に指定できます。
- 未指定の場合は L2TV の既定の screenshot フォルダに保存されます。

保存形式:

- PNG

ファイル名:

- 本日更新
  L2TV_Today_yyyymmddhhmmss.png

- 難易度表サマリー
  L2TV_Table_難易度表シンボル_yyyymmddhhmmss.png

同じ名前の画像がすでに存在する場合:

- 既存の画像を上書きせず、末尾に連番を付けて保存します。

保存完了:

- 「スクショ保存完了！」と表示されます。


11. 保存されるデータ
-------------------

L2TV は次回起動時にすぐ使えるように、以下の情報を保存します。

- score.db のパス
- song.db のパス
- Rival フォルダのパス
- スクショ保存先
- 選択した難易度表URL
- 選択した表示言語
- 選択したテーマ
- BP更新を本日更新に含める設定
- 前回の読み込み結果

保存データを消す方法:

1. メニューを開きます。
2. 「保存データを消す」を押します。

この操作で消えるのは L2TV 内の保存データだけです。
LR2 の score.db / song.db / Rival DB は削除されません。


12. 自動再読み込みについて
-------------------------

前回読み込んだ score.db / song.db に更新がある場合、
次回起動時や読み込み時に更新を検知して再読み込みできます。

これにより、LR2をプレイした後にL2TVを開くと、
新しく増えたランプ、BP更新、スコア更新などを確認しやすくなります。


13. テーマについて
-----------------

Light Aqua:
  水色ベースの明るいテーマです。

Geek Dark:
  黒基調の暗めのテーマです。

メニューから切り替えできます。


14. 安全性について
-----------------

L2TV は LR2 の score.db / song.db / Rival DB を読み込みますが、
DBファイルを書き換えません。

安全確認済みの内容:

- score.db / song.db / Rival DB は読み取り専用で扱います。
- LR2 のDBに対して UPDATE / DELETE / DROP TABLE などの書き込みSQLは実行しません。
- LR2 のフォルダやDBファイルを削除する処理はありません。
- LR2 のDBファイルを移動、リネーム、上書きする処理はありません。
- 任意の外部コマンドを実行する処理はありません。
- ユーザーが指定したスクショ保存先以外へ画像を勝手に保存しません。
- 画像保存時に同名ファイルがある場合は上書きしません。
- 画像ファイル名は危険な文字を取り除いてから保存します。
- ローカルAPIは外部サイトから勝手に使われにくいように制限しています。
- Electron の危険な機能は無効化しています。
- score.db / song.db / Rival DB を外部へアップロードする機能はありません。

つまり、L2TV は LR2 のデータを「見る」ためのツールであり、
LR2 のスコアデータを変更・削除するツールではありません。


15. 外部通信について
-------------------

L2TV は難易度表一覧や難易度表データを読み込むためにインターネットへ接続する場合があります。

接続する例:

- 難易度表一覧
- 選択した難易度表URL

送信しないもの:

- score.db
- song.db
- Rival DB
- LR2 のプレイヤーデータ
- ローカルDBの中身

DBの内容はローカルPC上で読み取って表示します。


16. Windows の警告について
-------------------------

個人配布の未署名アプリでは、Windows SmartScreen やウイルス対策ソフトが警告を出すことがあります。

これは「危険な処理がある」という意味ではなく、
署名や配布実績が少ないアプリでよく出る警告です。

不安な場合は、配布元に確認してから実行してください。


17. アップデート方法
-------------------

7z 版の場合:

1. L2TV を終了します。
2. 新しい L2TV-2.0.1-win-x64.7z を解凍します。
3. 古い L2TV フォルダと差し替えます。

注意:

- 念のため、L2TVを終了してから更新してください。
- LR2 の score.db / song.db は更新作業の対象ではありません。


18. アンインストール方法
-----------------------

旧インストーラー版についての重要事項:

- 1.0.1以前でインストール先にデスクトップなどの個人フォルダそのものを選んだ場合は、アンインストールを実行しないでください。

7z 版の場合:

1. L2TV のフォルダを削除します。
   Windows のアンインストール機能は使いません。

アプリ内保存データも消したい場合:

1. アンインストール前に L2TV を起動します。
2. メニューを開きます。
3. 「保存データを消す」を押します。
4. その後でアンインストールします。


19. よくある質問
----------------

Q. インターネットに接続していなくても使えますか？
A. 既に読み込んだ設定やローカルDBの確認はできます。難易度表一覧や難易度表データを新しく読み込む場合はインターネット接続が必要です。

Q. score.db や song.db は書き換えられますか？
A. いいえ。読み取り専用で扱うため、書き換えません。

Q. Rival DB は書き換えられますか？
A. いいえ。ライバル情報も読み取り専用で扱います。

Q. 難易度表を読み込まなくても使えますか？
A. はい。Player Data だけ表示できます。

Q. song.db は必須ですか？
A. 一部機能は score.db だけでも表示できますが、曲データ照合、段位、NO SONG判定などは song.db がある方が正確です。

Q. Rival フォルダは必須ですか？
A. いいえ。ライバル比較を使う場合だけ指定してください。

Q. 画像はどこに保存されますか？
A. メニューの「スクショ保存先」で指定したフォルダに保存されます。未指定の場合は既定の screenshot フォルダです。

Q. 配信やSNSに画像を載せてもいいですか？
A. 画像にはプレイヤー名、ライバル名、更新譜面などが含まれる場合があります。公開したくない情報がないか確認してから投稿してください。

Q. 難易度表一覧から表を選ぶと何が起きますか？
A. 選択した表の難易度表データを取得して、ローカルDBの譜面情報と照合します。score.db / song.db をアップロードするわけではありません。

Q. NSとは何ですか？
A. NO SONG の略です。難易度表にはあるが、読み込んだ song.db に曲が見つからない譜面です。

Q. BP更新とは何ですか？
A. 前回読み込み時より BP が減った譜面です。オプションをONにすると本日更新に含められます。

Q. スコア更新とは何ですか？
A. 前回読み込み時より EXスコアが伸びた譜面です。ランプ更新やBP更新とは別にスコア更新欄へ表示されます。

Q. Rankとは何ですか？
A. EXスコア率から判定した AAA / AA / A などの評価です。

Q. Folder Optionとは何ですか？
A. 開いているレベルフォルダ用の小型オプション表示です。並べ替えヘッダーを表示できます。
