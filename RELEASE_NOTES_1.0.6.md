# L2TV v1.0.6

難易度表の追加・選択・並べ替えを中心に、譜面一覧や本日更新の表示を改修しました。

これまでのURL貼り付け中心の方式から、難易度表一覧を検索して選択する方式へ変更しています。一覧にない表は、引き続きURLから追加できます。

## 難易度表まわりの主な変更

### 難易度表一覧から選択

- 難易度表一覧を専用画面で開けるようになりました。
- 一覧では `tag1` が `SP` の難易度表だけを取得します。
- 表名検索に対応しています。
- `tag2` を使用し、次のタグで絞り込めます。
  - General
  - Personal
  - Self-made Chart Only
- 各タグに該当する難易度表の件数を表示します。
- 一覧には `シンボル: 難易度表名` のみを表示し、探しやすくしました。

### 一覧にない難易度表の追加

- `table.html` または `header.json` のURLを指定して追加できます。
- URL追加時にヘッダー情報を取得し、シンボルと表名を自動表示します。
- ヘッダー情報を取得できなかった場合も、URL由来の仮名で追加できます。

### 追加済み難易度表の管理

- 「追加済み」から、読み込み対象の難易度表だけを確認できます。
- 追加済みの表をコンパクトな縦リストで表示します。
- マウスドラッグで直感的に並べ替えられます。
- 上下ボタンでも順番を微調整できます。
- 並べ替えた順番は保存され、アプリ内の難易度表表示順へ即時反映されます。

### 難易度表・譜面表示

- レベルフォルダをCLEAR LAMP / SCORE LAMPの横グラフとして表示します。
- 譜面一覧に次の項目を表示します。
  - Lamp
  - Rank
  - EX/Rate
  - BP
  - Play Count
  - Rival
- RankはAAA / AA / Aなどを色付きバッジで表示します。
- Folder Optionから、開いているレベル専用の並べ替えヘッダーを表示できます。
- Folder Closeから、スクロール位置にかかわらず開いているレベルを閉じられます。

## その他の主な更新

- FORCE RATEは発狂BMS難易度表と第二期Overjoy難易度表（https://lr2.sakura.ne.jp/overjoy.php）を参照します。
- 表外譜面を本日更新へ含める設定を追加しました。
- ランプ更新、BP更新、スコア更新を分けて表示します。
- 起動時に `score.db` / `song.db` の更新を検知できます。
- 前回読み込みから増えた打鍵回数とプレイ時間を表示します。
- 更新譜面が多い場合、本日更新画像を250件ごとに自動分割します。
- Rival比較、WIN率順 / LOSE率順 / 名前順の並べ替えに対応しました。
- 従来LR2IR互換DBとBMS-IR DBの自動判別・切り替えに対応しました。
- 日本語 / Englishと、Light Aqua / Geek Darkテーマに対応しました。

## 導入方法

1. `L2TV-1.0.6-win-x64.zip` をダウンロードします。
2. zipを任意の場所へ解凍します。
3. 解凍して作成された `L2TV` フォルダを開きます。
4. `L2TV.exe` を起動します。

インストールは不要です。zip内には、実行ファイルと構成データをまとめた `L2TV` フォルダが入っています。

## 難易度表の読み込み方

1. 画面右上の「メニュー」を開きます。
2. `score.db` と `song.db` を指定します。
3. 難易度表管理の「一覧を開く」を押します。
4. 読み込みたい難易度表へチェックを入れます。
5. 必要に応じて「追加済み」を開き、ドラッグで表示順を変更します。
6. 「表とランプを読み込む」を押します。

難易度表一覧や難易度表データの新規取得には、インターネット接続が必要です。

## 注意事項

- L2TVはLR2の `score.db` / `song.db` / Rival DBを読み取り専用で扱います。
- ローカルDBを外部へアップロードする機能はありません。
- 難易度表名は固有名詞として扱い、日本語 / English切り替え時も翻訳しません。
- Windows SmartScreenが警告を表示する場合があります。配布元とSHA256を確認してから実行してください。
- 削除するときはWindowsのアンインストーラーを使わず、解凍して作成されたL2TV専用フォルダだけをゴミ箱へ移動してください。
