L2TV 更新履歴
=============

1.0.4 - 2026/05/01
------------------

配布形式:

- 既定のWindowsビルドをインストーラーではなく zip 配布に変更しました。
- zip を解凍して L2TV.exe を起動すれば使えるようにしました。
- インストーラー作成は必要な場合だけ npm run dist:win:installer で実行する形にしました。

Player Data:

- score.db 内にIDがある場合、Player Data に名前とIDを表示するようにしました。
- GENOSIDE2018段位コースからSP段位を推測できない場合、score.db の grade_7 をSP段位表示のフォールバックとして使うようにしました。
- GENOSIDE2018段位コースの合格判定は EASY CLEAR を含めず、NORMAL CLEAR / HARD CLEAR / FULL COMBO のみを合格扱いにしました。
- score.db 内にプレイ時間項目がある場合、DB読み込み時の増加分をプレイ時間として本日更新に表示し、画像出力では打鍵数の横に表示するようにしました。

1.0.3 - 2026/05/01
------------------

IR関連機能の廃止:

- 譜面一覧の IR順位 列を削除しました。
- サマリーの IR 1位 / 2位 / 3位 / TOP10 保持数表示を削除しました。
- 本日更新画像や更新一覧から IR順位バッジ表示を削除しました。
- メニューの IR順位表示設定を削除しました。
- Player Data の LR2ID 表示を削除しました。
- song.db の ir_data を参照しないようにし、song.db は曲存在判定と段位解析にのみ使用するようにしました。

1.0.2 - 2026/05/01
------------------

緊急修正:

- インストーラーのインストール先変更を無効化しました。
- アンインストール時にL2TV専用フォルダ以外を削除しない安全確認を追加しました。
- 旧版のアンインストーラーを自動実行しないように、1.0.2からインストーラーIDを更新しました。

1.0.1 - 2026/05/01
------------------

英語対応:

- メニューから表示言語を 日本語 / English で切り替えられるようにしました。
- 初回起動時に表示言語を選択するメッセージを表示するようにしました。
- メニュー、ステータス、エラー表示、画像出力の主要ラベルを英語表示に対応しました。
- 英語版の readme_en.txt を追加しました。
- 難易度表名は固有名詞として扱い、言語切り替え時も翻訳しないようにしました。

ライバル機能:

- ライバルメニューに WIN率順 / LOSE率順 / 名前順の並べ替えを追加しました。
- WIN / LOSE の並べ替えを個数順ではなく割合順に変更しました。
- ライバル選択を変更しても、譜面一覧の並べ替え状態がリセットされないようにしました。
- 譜面一覧の Rival 列と Play Count 列の表示順を調整しました。

UI改修:

- 譜面一覧に Rank 列を追加し、AAA / AA / A などを色付きバッジで表示するようにしました。
- レベルフォルダを開いている間、Folder Close ボタンから下部で閉じられるようにしました。
- レベルフォルダを開いている間、Folder Option から小型の並べ替えヘッダーを表示できるようにしました。
- 譜面一覧のヘッダーや余白を調整し、横幅が広がりすぎないようにしました。
- ダークモード時のライバル表示やスコア内訳の文字色を調整しました。
- 本日更新に「スコア更新」欄を追加し、EXスコアが伸びた譜面を表示できるようにしました。
- SP段位表示は GENOSIDE2018 段位認定 の合格のみを参照するようにしました。
- 歴代Overjoyすべてに合格している場合は、Overjoy表示に特殊演出を出すようにしました。

安全性:

- ローカルAPIの接続元確認を強化し、ループバック以外からのアクセスを拒否するようにしました。
