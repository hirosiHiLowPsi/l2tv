L2TV 更新履歴
=============

2.1.0 - 2026/07/12
------------------

IR / ライバル:

- Stellaverse IRのIR順位・IR状況表示とStellaverse Rival機能を復活・整理しました。
- IR順位、IR状況、Stellaverse Rivalは現時点でStellaverse IRのみ対応しています。
- IR順位とIR状況をメニューから個別にON / OFFできるようにしました。
- IR表示を両方OFFにした場合、自分用IR順位の通信取得を行わないようにしました。

FORCE RATE:

- 前回読み込みからのFORCE RATEの増減を表示できるようにしました。
- FORCE RATE対象に新しく入った譜面（IN）と外れた譜面（OUT）を表示できるようにしました。
- FORCE RATE対象画像にも現在の称号バッジ、レート、前回比、IN / OUTを出力するようにしました。

UI:

- 上部操作ボタンをスクロール対応のアプリ内ツールバーへ整理しました。
- メニューを読み込み、難易度表、表示、その他の区分に整理しました。
- 難易度表の閉じた状態に譜面数、クリア率、プレイ率を表示するようにしました。
- 内訳カードの余白、文字位置、レベルフォルダの角表示を調整しました。
- 譜面一覧の横幅と通常行の色を整理し、画面を見やすくしました。

配布:

- バージョンを2.1.0へ更新しました。
- Windows向け配布形式は `L2TV-2.1.0-win-x64.7z` です。

2.0.1 - 2026/07/09
------------------

FORCE RATE:

- LR2IR Archive v3を参照して譜面定数を再計算しました。
- 対象表を発狂BMS難易度表、初代Overjoy、第二期Overjoyとして扱います。
- 初代Overjoyと第二期Overjoyに同じMD5の譜面がある場合は、第二期Overjoy側を優先して同一譜面として扱います。
- レート計算をスコア重視に整理し、クリアランプ係数とBEST20補正を使用しない方式に変更しました。
- FORCE RATEは、単曲FORCE上位50譜面と最高合格GENOSIDE2018 SP段位1件を合わせた最大51対象の単純平均で算出します。
- 発狂段位定数を、LR2IR Archive上の「プレイ人数に対する合格+AAA達成人数の割合」を基準にした値へ更新しました。
- 発狂段位コースはEXスコアが取得できる場合、通常譜面と同じスコア係数を段位定数にも反映します。
- GENOSIDE2018 Overjoyはスコア係数を掛けず、合格時点で段位定数満額として扱います。
- LR2の段位コースで `score.db` の `clear=2` として保存されるノーマル合格を、段位のNORMAL CLEAR相当として扱うようにしました。
- GENOSIDE2018 / GENOCIDE2018 の表記ゆれを吸収して段位コースを判定するようにしました。
- GENOSIDE2018 SP段位、Stella Skill Simulator 4th、Satellite Skill Analyzer 2nd の `.lr2crs` 由来コースhashを同梱し、段位推定のフォールバック照合に使うようにしました。
- st/sl段位の自動プロフィール更新時に、保存済みの古い段位ではなく新しく読み取った高い段位を優先するようにしました。
- st/sl段位コース名の解析を調整し、`st09` などの表記ゆれでも段位を拾いやすくしました。
- FORCE RATE対象51件を、現在の並び替え状態のままPNG画像として出力できるようにしました。
- 前回読み込み時からのFORCE RATE増減と、対象曲へ新しく入った曲（IN）・外れた曲（OUT）を確認できるようにしました。
- FORCE RATE対象画像にも、称号バッジ付きの現在レート、前回比、対象曲のIN / OUTを出力するようにしました。
- 公開用の譜面定数表とFORCE RATE説明画像を更新しました。

ライバル:

- Stellaverse Rival IDを入力し、公開スコアを取得して比較できる機能を追加しました。
- Rival DBを作成できない環境でも、EXスコア、スコアレート、BP、クリアランプ、WIN / LOSEを比較できます。
- Stellaverse IRの相手とローカルRival DBの相手を併用でき、登録したIDは次回読み込み時にも再取得します。
- StellaverseライバルのSP段位、FORCE RATE、称号名、称号バッジをライバル一覧へ表示するようにしました。
- 譜面一覧でライバルが未プレイの譜面は `No Play` のみ表示するようにしました。
- Stellaverse IRの非公開プロフィールでも、公開されているプレイヤー名とSP段位を取得して表示するようにしました。`🔒 非公開プロフィール` の注記自体は表示しません。

IR順位:

- Stellaverse IRから★・★★・st・sl譜面の順位と参加人数を表単位で取得し、譜面一覧に順位と上位率を表示する機能を復活させました。
- 同じプレイヤー・難易度表の取得結果は5分間キャッシュし、ライバル取得との重複アクセスも抑えるようにしました。
- 各難易度表にIR状況を追加し、1位・2位・3位・1～3位合計・4～10位合計を表示するようにしました。
- 譜面一覧の1位を金、2位を銀、3位を銅、4～10位を青緑系グラデーションで強調し、Play Countの右にIR順位を配置しました。

配布形式:

- 配布アーカイブ名を `L2TV-2.0.1-win-x64.7z` としました。
- 引き続き、解凍するとL2TV.exeと構成データが「L2TV」フォルダ内にまとまります。


2.0.0 - 2026/07/08
------------------

FORCE RATE:

- プレイヤーの総合指標として FORCE RATE を追加しました。
- 対象譜面、単曲レート、段位ボーナスをもとにレートを算出します。
- Player Data に FORCE RATE カードを表示し、対象譜面一覧を確認できるようにしました。
- 譜面定数は発狂BMS難易度表、初代Overjoy、第二期Overjoyを基準に扱います。
- GENOSIDE2018 Overjoy段位は、NORMAL CLEAR と HARD CLEAR の段位ランプ係数をどちらも1.00として扱います。

Stellaverse IR:

- score.db 内のIDを使って Stellaverse IR のプレイヤー名取得を試みる機能を追加しました。
- score.db仕様として 従来LR2IR互換 / Stellaverse IR を選択できるようにしました。
- 段位情報はIRサイトではなく、ローカルDBとsong.dbから取得する方針に整理しました。

難易度表選択:

- 難易度表URLを直接貼り付ける方式から、難易度表一覧から選択して読み込む方式に変更しました。
- 一覧では tag1 が SP の表だけを対象にし、tag2 の General / Personal / Self-made Chart Only で絞り込めます。
- 追加済みの表だけを表示し、ドラッグ操作でアプリ内の表示順を並べ替えられるようにしました。
- 一覧にない表はURL指定から追加でき、header情報を取得してシンボルと表名を表示します。

譜面一覧:

- 難易度フォルダをランプ分布グラフとして表示します。
- フォルダ内の達成状況に応じて、FC / HC以上 / NC以上 / EC以上 / 全プレイ済みでFLありの色でフォルダ行を表示します。

配布形式:

- 2.0.0以降の配布アーカイブを7z形式に変更しました。
- `L2TV-2.0.0-win-x64.7z` を解凍すると、L2TV.exeと構成データが「L2TV」フォルダ内にまとまります。


1.0.6 - 2026/06/20
------------------

難易度表一覧:

- 難易度表一覧URLを取得し、URL貼り付けではなく一覧から表を選択して読み込めるようにしました。
- 一覧では tag1 が SP の難易度表だけを表示します。
- tag2 を General / Personal / Self-made Chart Only の分類として使用します。
- 表名検索、タグごとの表示件数、追加済みの表だけを表示する機能を追加しました。
- 一覧表示を「シンボル: 難易度表名」に整理しました。
- DataTables形式や category / group など異なる列名の一覧データにも対応しました。

難易度表の追加・並べ替え:

- 一覧にない難易度表を table.html / header.json のURLから追加できるようにしました。
- URL追加時にヘッダー情報を取得し、シンボルと表名を自動表示します。
- 追加済み難易度表をコンパクトな縦リストで表示します。
- マウスドラッグと上下ボタンによる並べ替えに対応しました。
- 並べ替えた順番を保存し、アプリ内の難易度表表示順へ即時反映します。

UI:

- アプリのアイコンと画面左上のL2TVロゴを更新しました。
- 背景を単色に統一し、難易度表管理画面を整理しました。


1.0.5 - 2026/06/14
------------------

譜面一覧:

- レベルフォルダを CLEAR LAMP / SCORE LAMP の横グラフとして表示します。
- 譜面ごとに Lamp / Rank / EX-Rate / BP / Play Count / Rival を表示します。
- Rank は AAA / AA / A などを凡例に合わせた色付きバッジで表示します。
- フォルダを開いている間は Folder Close で画面下部から閉じられます。
- Folder Option から、開いているフォルダ専用の並べ替えヘッダーを表示できます。
- ライトモードとダークモードに対応しています。

本日更新:

- ランプ更新、BP更新、スコア更新を分けて表示します。
- 「表外譜面も更新楽曲として表示する」を有効にすると、読み込んでいる難易度表に含まれないローカル譜面も比較できます。
- score.db / song.db の更新を起動時に検知し、確認後に自動読み込みできます。
- 前回読み込みから増えた打鍵回数とプレイ時間を表示します。
- 本日更新をPNG画像として出力できます。
- 更新譜面が非常に多い場合は、250件ごとに複数のPNGへ分割して出力します。

ライバル:

- Rivalフォルダを読み込み、譜面ごとのWIN / LOSEと比較スコアを表示します。
- ライバルをWIN率順 / LOSE率順 / 名前順で並べ替えられます。
- 表示するライバルを個別選択、一括選択、一括解除できます。
- ライバル選択を変更しても、譜面一覧の並べ替え状態を維持します。

Player Data / 段位:

- プレイヤー名とIDを表示します。
- score.db仕様を 自動判別 / 従来LR2IR互換 / StellaverseIR / BMS-IR から選択できます。
- StellaverseIR選択時は、score.db内のIDを使ってStellaverseIRのプレイヤー名取得を試みます。
- SP段位とst/sl段位を表示します。
- ローカル段位判定は段位認定コース自体のクリア記録だけを参照し、構成する単曲のクリアランプは使用しません。
- GENOSIDE2018段位認定は NORMAL CLEAR / HARD CLEAR / FULL COMBO を合格として扱います。
- 段位情報はDBとsong.dbからオフラインで推定し、IRサイトから段位を補完取得しません。
- 2018 / 2012 / 2009 のOverjoyをすべて合格している場合は、専用表示になります。

表示言語:

- メニューから 日本語 / English を切り替えられます。
- 初回起動時に表示言語を選択できます。
- 難易度表名は固有名詞として翻訳せず、そのまま表示します。

配布形式:

- Windows向けzip版として配布します。
- zipを解凍すると、L2TV.exeと構成データが「L2TV」フォルダ内にまとまります。
- インストールは不要で、L2TV.exeから起動できます。
